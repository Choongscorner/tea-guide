## 2. 디자인 규칙 (기존 파일 기준)

```
배경: 다크 레드 그라디언트 (body::before)
카드 배경: rgba(255,255,255,0.94)
강조색: --red-dark: #7a1a1a  /  --gold: #B8860B
카드 그리드: repeat(auto-fill, minmax(340px,1fr))
사진: width:100%, height:200px, object-fit:cover, border-radius:8px
```

### 카드 구성 순서 (각 .rest-card 안)
1. `.rest-head` — 번호 뱃지 + 일본어명 + 음차 + 한자풀이 + 카테고리 뱃지
2. `<img class="food-photo">` — 음식 사진 (없으면 Wikimedia 동일 장르 사진)
3. `.special-box` — "왜 특별한가" 한 문단
4. `.info-section` > `.menu-list` — 메뉴 2~3개 (이름·음차·가격·설명)
5. `.info-section` > 영업시간·정기휴무·주소·교통
6. `.warn-box` — ⚠️ 주의사항 (현금, 매진, 예약 등)
7. `.enjoy-box` — 즐기는 법
8. `.map-btn` — 구글맵 링크 (`https://www.google.com/maps/search/{일본어명}+{주소}`)
