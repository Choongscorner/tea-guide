## 3. 차 종류 7가지 (등급 높은 순)

### ① 星野玉露 (ほしのぎょくろ) — 호시노 교쿠로
- **등급**: ⭐ 최고급 | **재배**: 강 차광, 야메시 호시노마을 한정
- **가격**: 50~120엔/g | **우리기**: 50~60°C / 10g / 40~60ml / 1분30초 / 3회
- **대표품**: 星野製茶園 ほしの秘園, 吉岡製茶 極選
- **쇼핑몰**: https://www.hoshitea.com/

### ② 玉露 (ぎょくろ) — 교쿠로
- **등급**: 고급~최고급 | **재배**: 강 차광 (90% 이상, 수확 20~30일 전)
- **가격**: 30~80엔/g | **우리기**: 50~60°C / 10g / 40~60ml / 1분30초 / 3회
- **야메**: 일본 교쿠로 생산 1위. 2007년 전국 경연 상위 26개 전부 야메산
- **대표품**: 牛島製茶 GI認証八女伝統本玉露, 古賀製茶本舗 玉露「匠」陽の雫

### ③ かぶせ茶 — 가부세차
- **등급**: 중~상급 | **재배**: 부분 차광 (수확 2주 전)
- **가격**: 20~35엔/g | **우리기**: 65~70°C / 5g / 120ml / 60~90초 / 3회
- **대표품**: いりえ茶園 奥八女かぶせ茶特上, 古賀製茶本舗 冠茶「翠」

### ④ 深蒸し煎茶 — 후카무시 센차 ⭐ 초보자 최추천
- **등급**: 중~상급 | **재배**: 비차광, 일반보다 2~3배 긴 증제
- **가격**: 13~25엔/g | **우리기**: 70~75°C / 5g / 200ml / **30~60초(짧게!)** / 2~3회
- **주의**: 세밀한 망(細目網) 급須 필수
- **대표품**: 古賀製茶本舗 深むし煎茶「薫」, 牛島製茶 深蒸し煎茶特上

### ⑤ 煎茶 — 센차
- **등급**: 중급 | **재배**: 비차광 (증제)
- **가격**: 10~20엔/g | **우리기**: 75~80°C / 4~5g / 200ml / 45~90초 / 3회
- **대표품**: 古賀製茶本舗 煎茶「薫」, JAふくおか八女 八女上煎茶

### ⑥ 玉緑茶・ぐり茶 — 타마료쿠차
- **등급**: 중급 | **재배**: 비차광 (비틀기 없어 둥근 잎 — 규슈 특산)
- **가격**: 15~30엔/g | **우리기**: 75~80°C / 5g / 200ml / 60초 / 3회 (투명 급須 추천)
- **대표품**: 新緑園 玉緑茶, 八女茶の里 八女玉緑茶

### ⑦ 釜炒り茶 — 카마이리차 (희귀: 일본 차 생산의 약 5%)
- **등급**: 중급 | **재배**: 비차광, 솥 볶기 (증제 아님)
- **가격**: 15~35엔/g | **우리기**: **85~90°C (가장 높음)** / 4g / 200ml / 60~90초 / 3회
- **향**: 釜香(카마코) — 고소한 아몬드 볶음 향
- **대표품**: 山輝園 特上釜炒り茶, 嬉野茶屋 嬉野釜炒り茶

## 6. 다기(茶器) 7종

| 일본어 | 읽기 | 용도 | 가격 |
|--------|------|------|------|
| 急須 | 큐스 | 일반 센차 찻주전자. 後카무시는 세밀한 망 필수 | 2,000~30,000엔 |
| 朱泥急須 | 슈데이 큐스 | 교쿠로 전용 60~100ml 붉은 점토 토기 | 5,000~80,000엔+ |
| 湯冷まし | 유자마시 | 물 식힘 그릇. 1회 이동 시 10~15°C 하강. 교쿠로 필수 | 1,500~8,000엔 |
| 湯飲み | 유노미 | 센차용 원통형 찻잔 120~200ml | 500~5,000엔 |
| 玉露茶碗 | 교쿠로 차완 | 교쿠로 전용 소형 30~50ml 찻잔 | 1,000~20,000엔 |
| 茶筒 | 차즈츠 | 차잎 보관 통 (이중 뚜껑 밀봉·방습) | 1,500~30,000엔+ |
| 茶杓 | 차샤쿠 | 차잎 계량 스쿠프. 1스쿠프 ≈ 2~3g | 300~3,000엔 |

### 핵심 기술 결정 (건드리면 안 되는 것들)

| 이슈 | 기존 방식 | 현재 방식 | 이유 |
|------|-----------|-----------|------|
| 배경 고정 | `background-attachment: fixed` | `body::before { position: fixed }` | Chrome/Edge 스크롤 jank |
| 유리 효과 | `backdrop-filter: blur()` 전체 | **nav 1개만** blur(10px) | 39개 적용 시 카드 내용 미렌더링 |

### 모바일 레이아웃 (@media max-width: 430px)
- `#teas table` → 카드 레이아웃 (`display:block` + `data-label` 속성)
- `#tasting table` → 카드 레이아웃 (동일 방식)
- nav → 가로 스크롤 (scrollbar 숨김)
- `.teaware-grid`, `.brew-grid` → 1열

### 한글 음차 원칙
- **반드시 일본어 발음 기준** (중국식 한자 독음 금지)
- 예시: `光安青霞園茶舗 (미츠야스 세이카엔 차포)` ← OK
- 잘못된 예시: `光安青霞園茶舗 (광안청하원차포)` ← X (중국식)

### store-name-origin 박스
일부 매장 카드에 `.store-name-origin` div로 이름 유래 설명 포함.
추가 시 기존 스타일 참조 (특별한 CSS 없음 — 일반 텍스트).
